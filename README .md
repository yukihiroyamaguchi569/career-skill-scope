# キャリアスキルスコープ

360°レーダーチャートを使って、人のキャリアとスキルを可視化するウェブアプリケーションです。仕事上のスキルだけでなく、趣味や関心事を含む人の全体的な能力を表現します。

## 概要

このアプリケーションでは、以下の特徴を持つレーダーチャートでスキルを可視化します：

- **スキルレベル**: 中心からの距離（0-10）で表現
- **ジャンル**: 中心からの角度（0-359度）で表現
- **時間軸**: 15歳から75歳まで5歳刻みでプロット
- **接続ルール**: 角度30°以内のスキルポイントは線で接続

## 12のスキルカテゴリ

スキルは以下の12の大カテゴリに分類されます：

1. **専門技術スキル**（0-29度）: プログラミング、医療技術、法律知識、会計など
2. **ビジネススキル**（30-59度）: マーケティング、財務管理、プロジェクト管理、営業など
3. **対人スキル**（60-89度）: コミュニケーション、リーダーシップ、チームワーク、交渉など
4. **知的スキル**（90-119度）: 分析力、批判的思考、問題解決、創造的思考など
5. **言語・コミュニケーション**（120-149度）: 外国語、文章作成、プレゼンテーション、ディベートなど
6. **音楽スキル**（150-179度）: 楽器演奏、歌唱、作曲、DJ、音楽理論など
7. **視覚芸術スキル**（180-209度）: 絵画、写真、デザイン、彫刻、映像制作など
8. **身体的スキル・スポーツ**（210-239度）: 球技、格闘技、陸上、水泳、ダンス、ヨガなど
9. **手工芸・DIYスキル**（240-269度）: 料理、裁縫、木工、ガーデニング、工作など
10. **デジタルスキル**（270-299度）: SNS運用、動画編集、ブログ運営、ゲーム開発など
11. **社会活動・ボランティア**（300-329度）: 地域活動、環境保護、教育支援、災害支援など
12. **自己啓発・マインドフルネス**（330-359度）: 瞑想、自己管理、時間管理、ストレス管理など

## 機能

- スキルデータの入力（年齢、カテゴリ、スキル名、レベル）
- レーダーチャートによる可視化
- 年齢スライダーによるデータ表示切替
- ローカルストレージによるデータ保存
- スキルリスト表示と削除機能
- ツールチップによる詳細情報表示
- レスポンシブデザイン対応

## 使用技術

- HTML5
- CSS3
- JavaScript (ES6+)
- D3.js（データ可視化）
- LocalStorage API（データ保存）

## アプリケーションの起動方法

### 1. 直接ブラウザで開く

`index.html` ファイルをブラウザ（Chrome、Safari、Firefoxなど）にドラッグ&ドロップするか、ブラウザから「ファイルを開く」でこのファイルを選択します。

### 2. ローカルサーバーを使用する方法（推奨）

ローカルサーバーを使うと、ブラウザのセキュリティ制限を回避でき、より安定して動作します。

#### a. VS Codeの拡張機能「Live Server」を使用

- VS Codeで拡張機能「Live Server」をインストール
- プロジェクトフォルダを開き、index.htmlを右クリックして「Live Serverで開く」を選択

#### b. Pythonを使用

```bash
cd /Users/Yukis_MacBook/git-local-repository/career-skiil-scope
python -m http.server
```

ブラウザで `http://localhost:8000` を開きます。

#### c. Node.jsのhttp-serverを使用

```bash
npm install -g http-server
cd /Users/Yukis_MacBook/git-local-repository/career-skiil-scope
http-server
```

ブラウザで表示されるURLを開きます。

## 使用方法

1. ブラウザでアプリケーションを開きます
2. 右側のフォームからスキルデータを入力します
   - 年齢を選択（15歳から75歳まで5歳刻み）
   - カテゴリを選択（12カテゴリから選択）
   - スキル名を入力
   - スキルレベルを設定（0-10）
3. 「スキルを追加」ボタンをクリックしてデータを保存
4. 左側のレーダーチャートに表示されるスキルを確認
5. 年齢スライダーを動かして異なる年齢のスキルを表示
